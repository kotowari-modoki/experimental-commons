# ライセンス: CC BY 4.0

このサイトで公開する文章コンテンツは、特記がない限り [Creative Commons Attribution 4.0 International (CC BY 4.0)](https://creativecommons.org/licenses/by/4.0/) の条件で再利用できる、という前提で運用する。

ただし、このページは法的助言ではない。判断に迷うケースでは、[公式のライセンス本文](https://creativecommons.org/licenses/by/4.0/legalcode) と各ページの注記を優先して確認してほしい。

## できること

CC BY 4.0 では、出典表示を行えば次の利用ができる。

- コピーして再配布する
- 編集、翻案、再構成する
- 商用・非商用を問わず利用する

## 必要なこと

再利用時には、少なくとも次の情報を示す。

- 著者名またはクレジット名
- 元ページへのリンク
- ライセンス名 `CC BY 4.0` とそのリンク
- 変更した場合は、その旨

## 表示例

### そのまま転載する場合

```text
出典: experimental-commons, "ページタイトル"
License: CC BY 4.0
https://kotowari-modoki.github.io/experimental-commons/...
```

### 編集して再利用する場合

```text
Based on experimental-commons, "ページタイトル" (CC BY 4.0).
Original: https://kotowari-modoki.github.io/experimental-commons/...
Modified for clarity and formatting.
```

## 適用範囲

現時点では、README と `package.json` に `CC BY 4.0` の記載がある。このページでは、その方針をサイト上の文章コンテンツについて明示している。

一方で、以下はこのページの説明だけでは扱いを確定できない場合がある。

- 引用している第三者の文章、画像、図表
- 埋め込みコンテンツや外部サービス由来の素材
- 別ライセンスを個別に明記したコードやアセット

そのため、各ページに個別注記がある場合はそちらを優先する。第三者コンテンツを再利用するときは、その元ライセンスまで遡って確認してほしい。

## このサイトでの基本姿勢

- 再利用しやすいことを優先する
- 出典を追えるように一次情報へのリンクを残す
- 不確かな知識は不確かだと明記する

この方針は、完成した成果物だけでなく、途中経過の知見も公共財として流通しやすくするためのものだ。
